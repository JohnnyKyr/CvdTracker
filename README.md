# CVDtrack
Web development university project 2021-2022

### Authors
- [Xaris Kap](https://github.com/xariskap)
- [Johnny Kyr](https://github.com/JohnnyKyr)

The purpose of this application is to track the places someone has visited where a positive to covid individual has been there
